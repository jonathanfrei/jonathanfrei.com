New homepage

Hi, my name is Jonathan Frei. You can email me at hi [@] jonathan frei dot com or reach me on Twitter at @jonathanfrei although I don't really use it anymore.

For this new site, I just make the HTML files in Github, and push it out onto the internet. Everything can fit into a single file. I can link to whatever I want and add the images in line or through hot linking as needed.

Is this a better way to publish on the web? Probably not. But it is fun! It's like driving a stick shift...getting closer to the metal.

Earlier iterations of this site are still available online. I published V1 on Blogger from 2009 to 2010; V2 on Tumblr from 2010 to 2014; V3 on a self-hosted Wordpress from 2014 to 2022 (online version is now a static copy hosted in an S3 bucket). The current version lives in Github and is distributed through Cloudflare.
